🚀初次快速上手 SOP
1鏡頭安裝安全守則
更換鏡頭前必須關閉主電源。移除孔蓋後插入並順時針旋至「喀嚓」聲，務必鎖緊螺絲防止掉落。
2電源電壓與亮度限制
全亮度輸出必須接 AC 200–240V。若接 AC 100–120V，亮度將自動受限於約 12,000 lm（AC 200V 為關鍵因素）。
3鏡頭校正（更換後必做）
為確保馬達精準感應物理極限，更換鏡頭後務必執行校正流程。 路徑：[6. 投影機設置] → [鏡頭] → [鏡頭校正]
4系統出廠初始化（強烈建議）
建議初次使用執行「出廠重置」，清空殘留設定，確保影像正常。 路徑：[6. 投影機設置] → [初始化] → [出廠重置]
⚠️深度故障排除與警報應對
Q：出現 U 開頭警報代碼？（自我診斷） • U12 / U13：進/排氣溫度過高。請立即清潔濾網並確認進氣孔無遮擋。
• U15：排氣溫度過高。請確認機器後方散熱空間（至少留 50cm）。
• U081：AC 輸入電壓過低（低於 90V）。請檢查供電系統負載。
Q：出現 F 開頭代碼？（硬體故障） 屬於內部零件異常（如 F1：風扇故障、F2：光源驅動異常）。請記錄代碼後，拔掉電源等候 5 分鐘再重試。若持續出現，請聯絡技術人員。
Q：畫面亮度明顯不足？ 1. 確認電壓：是否誤接 110V？
2. 操作模式：是否誤設為 [ECO] 或 [靜音]？
3. 亮度控制：檢查是否開啟 [恆定模式]，這會導致初始亮度受限。
4. 快門：確認機械快門（Shutter）是否處於半開狀態或誤設淡入時間。
Q：鏡頭無法位移（Lens Shift）？ 1. 檢查 [6. 投影機設置] → [鏡頭] → [鏡頭鎖] 是否設為「開」。
2. 執行 [鏡頭校正]，讓機器重新感應馬達極限點。
🔧進階維護參考
A黑平衡色彩校準（偏色修正）
若暗場顯示不純時執行。執行時必須蓋上鏡頭蓋並確保無任何漏光。 路徑：[1. 圖像] → [進階影像調節] → [黑平衡]
B鏡頭安裝與斷電保護（詳細版）
更換鏡頭前必須關閉主電源。插入鏡頭順時針旋至「喀嚓」聲後，務必鎖緊側邊安全螺絲。開機後執行 [鏡頭校正] 讓馬達感應極限位置，否則無法進行幾何調整。
B電源與亮度核對
若使用 AC 100–120V 電源，亮度會自動降至約 12,000 lm（約 1/3）。全亮度 30,500 lm 必須接 AC 200–240V。 核對指標：[6. 投影機設置] → [狀態] → [AC 電壓]
C光學中心與對焦平衡
除了選單內的 Focus 調整，若畫面邊緣模糊，請手動旋轉鏡頭上的「外圍焦點調整環」進行對焦平衡。建議在熱機 30 分鐘後執行。
D黑平衡校準（Black Balance）
環境光源改變或黑色發紫/發綠時，請務必蓋上鏡頭蓋後執行。此操作會校準投影機在無訊號下的暗部基礎色彩。 路徑：[1. 圖像] → [進階影像調節] → [黑平衡]
E4K 訊號與 EDID 手動握手
若電腦無法抓到 4K 訊號，請至選單手動設定 EDID 模式。 路徑：[5. 顯示方式選項] → [輸入端子設定] → [EDID 選項] → 切換為 [4K/60p] 或 [USER]
F幾何矯正與 4K 驅動關係
當開啟 [自由網格] 或特定幾何修正時，[QUAD PIXEL DRIVE] 可能會被強制固定為 [開]。
G數據備份（Data Cloning）
完成校正後，建議使用 USB 隨身碟備份所有用戶數據。此備份可用於快速換機還原幾何與色彩設定。 路徑：[6. 投影機設置] → [數據克隆] → [寫入 USB]
H濾網維護與計時重置
清潔濾網後，必須在選單中重置濾網計時器，否則系統會持續報警（U12/U13）並可能限制亮度以防過熱。 路徑：[6. 投影機設置] → [狀態] → [濾網計時器]
